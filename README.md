## :information_source: How to use

To clone and run this application, you will need [Git](https://git-scm.com), [Node.js][nodejs] e [Yarn][yarn] installed on your computer.

Through command line:

```bash
# Clone this repositori
$ git clone https://github.com/DanielNeris/GoStack-GoBarber.git
# Go to backend
$ cd backend
# Install dependences
$ yarn install
# Start the servidor
$ yarn dev
# Exit backend
$ cd ..
# Go to frontend
$ cd frontend
# Install dependences
$ yarn install
# Start the frontend
$ yarn start
```

[nodejs]: https://nodejs.org/
[yarn]: https://yarnpkg.com/
[vc]: https://code.visualstudio.com/
